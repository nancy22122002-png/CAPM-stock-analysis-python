# CAPM-stock-analysis-python
Project Overview

This project applies the Capital Asset Pricing Model (CAPM) to analyze stock performance and evaluate the relationship between risk and expected return. Using Python and financial data analysis techniques, the project examines how individual stocks behave relative to the overall market and estimates their expected returns based on systematic risk.

The analysis includes data preprocessing, exploratory data analysis, return calculation, beta estimation, and financial visualization to better understand market dynamics and investment performance.

Objective

The main objectives of this project are:

To analyze historical stock market data and understand price trends.

To calculate daily stock returns and evaluate market volatility.

To measure Beta values to determine how sensitive stocks are to market movements.

To apply the Capital Asset Pricing Model (CAPM) to estimate expected returns.

To visualize stock performance and compare it with the market index.

Tools and Technologies

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

These tools were used for data analysis, financial modeling, and visualization of stock performance.

Methodology
1. Data Collection

Historical stock price data was collected and loaded into a Python environment for analysis.

2. Data Preprocessing

Data cleaning and formatting were performed to ensure accurate analysis, including handling missing values and organizing time-series data.

3. Exploratory Data Analysis (EDA)

Initial analysis was conducted to understand trends, price distributions, and stock behavior over time.

4. Daily Return Calculation

Daily returns were calculated to measure percentage changes in stock prices and analyze volatility.

5. Beta Calculation

Beta values were computed to measure the systematic risk of stocks relative to the market index.

6. CAPM Implementation

The Capital Asset Pricing Model was used to estimate expected stock returns based on risk-free rate, beta, and market return.

7. Data Visualization

Graphs and charts were generated to visualize stock performance, market trends, and investment growth.

Key Results and Insights

Identified the risk exposure of stocks using Beta values.

Estimated expected stock returns using CAPM.

Analyzed the relationship between stock returns and market returns.

Demonstrated investment growth through portfolio simulations.

Provided insights into the risk-return trade-off in financial markets.

Conclusion

This project demonstrates how financial theory and data analysis can be combined to evaluate investment risk and expected returns. By implementing CAPM and analyzing historical stock data, the project highlights how investors can assess whether a stock’s return justifies its level of risk.

The project also showcases practical skills in financial modeling, quantitative analysis, and Python-based data analysis, which are valuable in finance, investment research, and financial analytics roles.
